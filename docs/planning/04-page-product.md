# Page Content — Product / Shop Page

## Route

`/product`

## Page Purpose

The product page is the focused shop page. Since there is one main product, this page must make that product feel valuable and complete. It should explain quality, freshness, presentation, and usage occasions before moving the user to availability/order.

## Primary CTA

## Secondary CTA
