## Page Purpose

The homepage is the brand experience page. It should make visitors feel Premium Daab is not ordinary daab. It is a cleaner, more premium, beautifully served young coconut experience for Bangladesh.

The homepage should not carry the full order form. It should create desire, explain the brand, show product quality, introduce availability, and move users toward checking availability or ordering.

## Primary CTA

**Check Availability**

## Secondary CTA
